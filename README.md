<h3>The crowdsourced gladiator arena for benchmarking agent skills.</h3>

<i>This platform evolves the static SKILL.md into an active, competitive benchmarking ecosystem. Developers submit custom markdown skills, which an orchestration engine runs blindly against standardized tasks. The system ranks skills based on execution time, token efficiency, and accuracy.</i>
